## 프로젝트 한 줄 요약

한국어 AI Gateway용 **PII(개인정보) 감지 · 마스킹 가드레일** 시스템.
정규식(Regex) → 사전(Dictionary) → NER(딥러닝) → 컨텍스트 스코어 → 정책 → 마스킹 의 M1~M9 파이프라인으로 구성.

## 카테고리별 상세 안내

### 1. `korean_pii_guardrail_v0_2/` — 메인 제품

| 하위 폴더 | 역할 |
|-----------|------|
| `src/pii_guardrail/` | 핵심 라이브러리 (M1~M9 파이프라인 구현체) |

| `docs/` | 설계 명세서 00~18번 |

### 2. `PII/` — 연구·실험 레이어

초기 아키텍처 탐색 단계의 4-way 비교 실험 (Layer 0~4).
v0.2 제품 개발 전 성능 벤치마크 데이터 보관.

| 하위 폴더 | 역할 |
|-----------|------|
| `layer_0/` | 정규식 기반 감지기 (Layer 0) 원형 |
| `layer_1~4/` | 각 레이어 실험 결과 JSON |
| `fuzzer/` | 한국어 PII 데이터 생성기 (이름/주소/처방전/계좌/거래) |
| `evaluation/` | 4-way 성능 비교 스크립트 + 그래프 생성 |
| `datasets/` | 정상 텍스트 + 공격 payload 데이터셋 |
| `results/` | Phase 1~6 실험 결과 및 그래프 |
| `config/` | Docker Compose + 실험 설정 |

→ 상세 목차: [`PII/목차.md`](PII/목차.md)
