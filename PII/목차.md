## 4-way 비교 실험 개요

| 레이어 | 방식 | 결과 위치 |
|--------|------|---------|
| Layer 0 | 정규식(Regex) 기반 감지 | `layer_0/`, `results/` |
| Layer 1 | Claude API (직접 호출) | `layer_1/results_L1_*.json` |
| Layer 2 | LiteLLM 게이트웨이 | `layer_2/results_L2_*.json` |
| Layer 3 | (예약) | `layer_3/` |
| Layer 4 | 커스텀 가드레일 (Anthropic API 통합) | `layer_4/custom_guardrail.py` |
