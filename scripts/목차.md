## 스크립트 상세

### `build_pptx_v5.py`

**역할:** `python-pptx` 기반으로 발표 슬라이드 v5를 프로그래밍 방식으로 생성  
**출력:** `presentations/AI_Gateway_가드레일_프로젝트_발표_v5.pptx`

```bash
python scripts/build_pptx_v5.py
```

### `run_eval_pipeline.sh`

**역할:** `PII/evaluation/` 의 평가 스크립트들을 순서대로 실행하는 자동화 셸  
**실행 순서:** sample → baseline → L0 compare → 4-way compare → final

```bash
bash scripts/run_eval_pipeline.sh
```
