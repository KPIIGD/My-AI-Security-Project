## 스크립트 상세

### `build_pptx_v5.py`

**역할:** `python-pptx` 기반으로 발표 슬라이드 v5를 프로그래밍 방식으로 생성  
**출력:** `presentations/AI_Gateway_가드레일_프로젝트_발표_v5.pptx`

```bash
python scripts/build_pptx_v5.py
```

### `run_eval_pipeline.sh`

**역할:** `PII/evaluation/` 의 평가 스크립트들을 순서대로 실행하는 자동화 셸  
**실행 순서:** sample → baseline → L0 compare → 4-way compare → final

```bash
bash scripts/run_eval_pipeline.sh
```

### `fetch_slack_logs.py`

**역할:** Slack Web API로 채널 대화를 조회해 `../슬랙/<날짜>.md` 로 저장 (의존성 없는 stdlib만 사용)  
**입력:** 환경변수 `SLACK_BOT_TOKEN` (scope: `channels:history`, `channels:read`, `users:read`)  
**출력:** `슬랙/<날짜>.md` (`.gitignore` 처리되어 공개 레포에 커밋되지 않음)

```bash
# PowerShell:  $env:SLACK_BOT_TOKEN = "xoxb-..."
export SLACK_BOT_TOKEN=xoxb-...
python scripts/fetch_slack_logs.py --channel general,random --date 2026-06-18
```
