## `src/pii_guardrail/` — 핵심 라이브러리

### 메인 모듈

| 파일 | 역할 | 파이프라인 단계 |
|------|------|---------------|
| `pipeline.py` | `GuardrailPipeline` — M1→M9 통합 진입점 | M9 |
| `preprocess.py` | 한국어 정규화, 유니코드 정리 | M1 |
| `regex_detectors.py` | 정규식 기반 PII 감지 (RRN, 전화번호, 이메일 등) | M2 |
| `korean_boundary.py` | 한국어 형태소 경계 보정, 조사 분리 | M3 |
| `dictionary_detectors.py` | 사전 기반 감지 (병원명, 기관명 등) | M4 |
| `context_scorer.py` | 컨텍스트 기반 신뢰도 스코어링 | M4/M5 |
| `span_resolver.py` | 중복·겹침 span 병합 및 해소 | M6 |
| `policy.py` | 정책 라우터 (BLOCK/MASK/ALLOW 결정) | M7 |
| `masker.py` | 마스킹 실행 (토큰 대체) | M7 |
| `audit_logger.py` | HMAC 서명 감사 로그 (JSONL sink) | M8 |
| `schema.py` | Pydantic 스키마 (요청/응답/span) | 공통 |
| `interfaces.py` | 추상 인터페이스 | 공통 |
| `enums.py` | 엔티티 타입 열거형 | 공통 |
| `detector_config.py` | 감지기 설정 로딩 | 공통 |
| `dictionary_loader.py` | 사전 파일 로딩 | M4 |
| `validators.py` | 입력 유효성 검사기 | 공통 |
| `evaluation_harness.py` | 평가 프레임워크 | 평가 |

### NER 서브모듈 (`ner/`)

| 파일 | 역할 |
|------|------|
| `base.py` | NER 추상 베이스 클래스 |
| `owner_wrapper.py` | KLUE-RoBERTa-Large-v3 모델 래퍼 (실제 추론) |
| `finetuned_wrapper.py` | 파인튜닝 모델 래퍼 |
| `mock_ner.py` | 테스트용 Mock NER |

## `configs/` — 설정 파일

| 파일 | 역할 |
|------|------|
| `entities.yaml` | 감지 엔티티 타입 정의 전체 목록 |
| `detectors.yaml` | 각 감지기 활성화 여부·임계값·옵션 |

| `dictionaries.yaml` | 사전 파일 경로 매핑 |
| `josa_rules.yaml` | 한국어 조사 목록·처리 규칙 |
